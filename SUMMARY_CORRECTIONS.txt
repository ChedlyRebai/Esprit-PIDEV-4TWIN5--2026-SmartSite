│ 📡 NOUVEAUX ENDPOINTS                                                          │
└────────────────────────────────────────────────────────────────────────────────┘

  1. GET /materials/:id/site-details
     └─ Récupère les détails du site avec GPS et localisation
     └─ Réponse: { siteId, siteName, siteAddress, siteLocalisation, coordinates }

  2. GET /materials/:id/aggregate-stats?siteId=SITE_ID
     └─ Récupère les statistiques des mouvements
     └─ Réponse: { totalEntries, totalExits, netFlow, totalAnomalies }

  3. GET /materials/diagnostic/sites
     └─ Diagnostic des sites assignés aux matériaux
     └─ Réponse: { totalMaterials, materialsWithSites, siteCheckResults }
