## 🔗 Liens Rapides

### Commandes Essentielles

**Démarrer Materials Service**:
```bash
cd apps/backend/materials-service
npm start
```

**Démarrer Frontend**:
```bash
cd apps/frontend
npm run dev
```

**Vérifier le système**:
```powershell
# Windows
.\check-system.ps1

# Linux/Mac
bash check-system.sh
```

**Tester les endpoints**:
```bash
curl http://localhost:3009/api/materials
curl http://localhost:3009/api/materials/predictions/all
```

### Fichiers Importants

**Configuration**:
- `apps/frontend/vite.config.ts` - Proxy Vite
- `apps/backend/materials-service/.env` - Port 3009
- `apps/backend/api-gateway/.env` - URL Materials Service

**Code**:
- `apps/frontend/src/app/pages/materials/Materials.tsx` - Affichage
- `apps/backend/materials-service/src/materials/services/stock-prediction.service.ts` - Calcul
- `apps/backend/materials-service/src/materials/materials.controller.ts` - Endpoints
