│ 🚀 DÉMARRAGE RAPIDE                                                            │
└────────────────────────────────────────────────────────────────────────────────┘

  1. Vérifier que le backend compile:
     $ cd apps/backend/materials-service
     $ npm run build
     ✅ Succès

  2. Vérifier que le frontend compile:

     $ npm run build
     ✅ Succès

  3. Démarrer les services:
